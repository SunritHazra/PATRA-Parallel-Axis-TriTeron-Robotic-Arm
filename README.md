# PATRA-Parallel-Axis-TriTeron-Robotic-Arm
A robot with legs of a parallel axis tripteron (or tri-teron) and a 6 dof precision robotic arm on the top to make it functional. This is a kinematically linear system. Note that i have never seen anyone adding a robotic arm to this, so the idea is itself original. Plus, I will be using ball screw for the main carriage.
